Open-source LED Poi

The aim of this project is to create an open-source framework for LED Poi that can be used to create a variety of Poi from simple colour-changing effects to advanced customisable LED pixel poi.

Goals

The poi will be easy and cheap to make. They will be made using easily sourced materials and require no expensive tools to produce. The electronics will use open platforms like Arduino or Digispark.

The software will be open-source and hackable. Available via GitHub.

The colours and patterns should be easy to customise for non-technical users.
